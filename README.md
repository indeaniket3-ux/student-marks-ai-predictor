# 🎯 Student Marks AI Predictor

This is a beginner Machine Learning project using Linear Regression.

## 🔧 Tech Used
- Python
- NumPy
- Pandas
- Scikit-Learn
- Matplotlib

## 📊 What It Does
Predicts student marks based on hours studied and sleep hours.
